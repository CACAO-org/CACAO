148 // STREAM LOADING POLICY FLAGS
149 // These flags modify the default stream load policy
150 // Default load policy: FORCE flags = 0, SKIPSEARCH flags = 0, UPDATE flags = 0
151 //
152 // FORCE flags will force a location to be used and all downstream locations to be updated
153 // if the FORCE location does not exist, it will fail
154 // only one such flag should be specified. If several force flags are specified, the first one ((a) over (b)) will be considered
